⚙️ Setup required

This booking system uses a Google Sheet as its database. You need to deploy the Apps Script backend once, then paste the Web App URL below.

  1. Open Google Sheets → create a new spreadsheet
  2. Go to Extensions → Apps Script
  3. Paste the contents of BookingBackend.gs and save
  4. Click Deploy → New deployment
  5. Type: Web app · Execute as: Me · Access: Anyone
  6. Copy the Web App URL and paste it below
